1. Planlægning og Opsætning af Projektet

Trin 1: Opsætning af Visual Studio

  1. Download og installer Visual Studio Community Edition.
  2. Under installation, vælg workloads "ASP.NET and web development" og "Azure development".

Trin 2: Oprettelse af et nyt projekt

  1. Åbn Visual Studio og vælg "Create a new project".
  2. Vælg "ASP.NET Core Web Application" og klik "Next".
  3. Giv projektet et navn (fx "BirthdayReminder") og vælg en placering.
  4. Vælg ".NET Core" og "ASP.NET Core 5.0 (eller nyere)" som framework. Vælg "Web Application (Model-View-Controller)" skabelonen og klik "Create".

Trin 3: Installation af nødvendige NuGet-pakker

  1. Åbn "NuGet Package Manager" ved at højreklikke på projektet i Solution Explorer og vælge "Manage NuGet Packages".
  2. Søg og installer følgende pakker:
    • Microsoft.AspNetCore.Identity.EntityFrameworkCore
    • Microsoft.EntityFrameworkCore.SqlServer
    • Microsoft.EntityFrameworkCore.Tools

2. Brugerautentificering og -autorisation

Trin 4: Opsætning af brugerautentificering

  1. Tilføj følgende kode i Startup.cs under ConfigureServices for at konfigurere Identity:

    csharp
    services.AddDbContext<ApplicationDbContext>(options =>
    options.UseSqlServer(Configuration.GetConnectionString("DefaultConnection")));
services.AddDefaultIdentity<IdentityUser>(options => options.SignIn.RequireConfirmedAccount = true)
    .AddEntityFrameworkStores<ApplicationDbContext>();

  2. Konfigurer databaseforbindelsen i appsettings.json:

    json
    "ConnectionStrings": {
    "DefaultConnection": "Server=(localdb)\\mssqllocaldb;Database=BirthdayReminder;Trusted_Connection=True;MultipleActiveResultSets=true"
}

  3. Kør Add-Migration InitialCreate og Update-Database i Package Manager Console for at oprette databasen.

Trin 5: Oprettelse af brugerroller

  1. Opret en klasse RoleInitializer for at oprette standardroller:

    csharp
    public class RoleInitializer
{
    public static async Task InitializeAsync(UserManager<IdentityUser> userManager, RoleManager<IdentityRole> roleManager)
    {
        string adminEmail = "admin@example.com";
        string password = "Admin123!";
        if (await roleManager.FindByNameAsync("admin") == null)
        {
            await roleManager.CreateAsync(new IdentityRole("admin"));
        }
        if (await userManager.FindByNameAsync(adminEmail) == null)
        {
            IdentityUser admin = new IdentityUser { Email = adminEmail, UserName = adminEmail };
            IdentityResult result = await userManager.CreateAsync(admin, password);
            if (result.Succeeded)
            {
                await userManager.AddToRoleAsync(admin, "admin");
            }
        }
    }
}

  2. Kald RoleInitializer.InitializeAsync i Configure-metoden i Startup.cs.

4. CRUD-operationer og Business Logic

Trin 7: Implementering af CRUD-operationer

  1. Opret en controller BirthdaysController:
    csharp
    public class BirthdaysController : Controller
{
    private readonly ApplicationDbContext _context;

    public BirthdaysController(ApplicationDbContext context)
    {
        _context = context;
    }

    public async Task<IActionResult> Index()
    {
        return View(await _context.Birthdays.ToListAsync());
    }

    public IActionResult Create()
    {
        return View();
    }

    [HttpPost]
    public async Task<IActionResult> Create(Birthday birthday)
    {
        if (ModelState.IsValid)
        {
            _context.Add(birthday);
            await _context.SaveChangesAsync();
            return RedirectToAction(nameof(Index));
        }
        return View(birthday);
    }

    // Implementer Edit, Delete, Details metoder på lignende måde
}
